# Batocera-RaspberryPi3B
## Build Steps (for thumb drive)
### Set Up Sound
System Settings -> Audio Output->Build-In Audio Digigtal Stereo(HDMI)
### Set Time
System Settings -> Time Zone -> America/New_York
Show Clock in 12-Hour Format = Yes
### Network
Network Settings->Enable Wifi->ID and Password
Hostname = BATOCERA3td
### Upgrade to Beta
Updates and Downloads->Update Type = Butterfly
Start Update
